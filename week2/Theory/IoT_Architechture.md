**4-Layer IoT Architecture**
1. Application Layer

Example: Mobile App

Function:
Provides user interface and allows users to access IoT services.

2. Processing Layer

Example: Cloud Server (AWS IoT)

Function:
Stores, processes, and analyzes data. Makes decisions and provides intelligence.

3. Network Layer

Example: Wi-Fi Router

Function:
Transmits data between perception devices and processing systems.

4. Perception Layer

Example: DHT11 Sensor

Function:
Collects data from the physical environment using sensors and devices.

Data Flow: Bottom to Top
