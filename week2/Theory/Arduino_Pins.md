| **Pin Type**       | **Explanation**                                                                                                                                                               | **Example Device**                       | **Real IoT Use Case**                                                                                                                 |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| **Digital Input**  | Reads two states: **HIGH (1)** or **LOW (0)** from sensors or switches.                                                                                                       | Push Button, PIR Motion Sensor           | A **smart doorbell** detects when a button is pressed and sends a notification.                                                       |
| **Digital Output** | Sends **HIGH** or **LOW** signals to control devices.                                                                                                                         | LED, Relay Module, Buzzer                | A **smart home system** turns an LED or relay ON/OFF to control lights or appliances.                                                 |
| **Analog Input**   | Reads varying voltage levels (0–5 V) using the ADC and converts them into digital values (0–1023).                                                                            | LDR (Light Sensor), Soil Moisture Sensor | A **smart irrigation system** measures soil moisture and waters plants automatically.                                                 |
| **PWM Output**     | Produces a Pulse Width Modulation signal to simulate analog output and control power. Available on pins **3, 5, 6, 9, 10, and 11**.                                           | Servo Motor, DC Motor, LED               | A **smart fan controller** adjusts fan speed based on room temperature.                                                               |
| **I2C / SPI Pins** | Communication pins used to connect multiple sensors and modules with high-speed data transfer. **I2C:** SDA (A4), SCL (A5). **SPI:** SS (10), MOSI (11), MISO (12), SCK (13). | OLED Display, RTC Module, SD Card Module | A **weather monitoring station** reads data from multiple sensors and displays it on an OLED screen while logging data to an SD card. |
